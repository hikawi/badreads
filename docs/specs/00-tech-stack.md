
# Tech Stack & Libraries

## Frontend
- React.js
  - React Router DOM - Client-side routing
  - Redux Toolkit - State management
  - Axios - HTTP client
  - Material UI - Component library
  - React Query - Server state management
  - React Hook Form - Form handling
  - Jest & React Testing Library - Testing

## Backend 
- Express.js
  - Node.js runtime
  - MongoDB - Database
  - Mongoose - ODM
  - JWT - Authentication
  - bcrypt - Password hashing
  - Express Validator - Request validation
  - Morgan - HTTP request logger
  - Helmet - Security headers
  - CORS - Cross-origin resource sharing
  - Winston - Logging
  - Jest - Testing

## Development Tools
- TypeScript - Type safety
- ESLint - Code linting
- Prettier - Code formatting
- Husky - Git hooks
- Docker - Containerization
- GitHub Actions - CI/CD

## Package Manager
- npm/yarn

## Deployment
- Frontend: Vercel - React hosting (free tier)
- Backend: Railway/Render - Node.js hosting (free tier)
- Database: MongoDB Atlas - Cloud database (free 512MB)
- CDN: Built into Vercel
- DNS: Cloudflare - Domain management (free)

This gives a completely free, production-ready setup for the initial MVP version.
We can migrate to paid services as our app grows, lol hope that happens!!!.
